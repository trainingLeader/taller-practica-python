# Documentación Taller Practico

# Ejercicio 1: Clasificador de números

## Objetivo

Crear un programa que solicite un número entero y determine si es:

- Positivo.
- Negativo.
- Cero.
- Par.
- Impar.

## Requisitos

El programa debe:

1. Pedir un número entero.
2. Evaluar si el número es positivo, negativo o cero.
3. Evaluar si el número es par o impar.
4. Mostrar los resultados en pantalla.

# Ejercicio 2: Contador con `while`

## Objetivo

Crear un programa que imprima los números del `1` al `20` .

## Requisitos

El programa debe:

1. Iniciar un contador en `1`.
2. Repetir mientras el contador sea menor o igual a `20`.
3. Mostrar cada número.
4. Aumentar el contador en cada repetición.

# Ejercicio 3: Números pares con `for`

## Objetivo

Crear un programa que muestre los números pares del `2` al `50`.

## Requisitos

El programa debe:

1. Usar un ciclo `for`.
2. Recorrer los números del `2` al `50`.
3. Mostrar solo los números pares.

# Ejercicio 4: Tabla de multiplicar

## Objetivo

Crear un programa que solicite un número y muestre su tabla de multiplicar del `1` al `10`.

## Requisitos

El programa debe:

1. Pedir un número al usuario.
2. Usar un ciclo `for`.
3. Multiplicar el número por los valores del `1` al `10`.
4. Mostrar cada resultado.

# Ejercicio 7: Menú básico con `match`

## Objetivo

Crear un menú que permita al usuario seleccionar una opción usando `match`.

## Requisitos

El programa debe mostrar este menú:

```
1. Saludar
2. Mostrar mensaje motivacional
3. Mostrar autor del programa
4. Salir
```

El programa debe ejecutar una acción según la opción seleccionada.
